# 日常送礼维护工作流

简短说明：新增或补充一个联络（送礼）对象时，只需两步：在角色数据里添加角色信息，且通过软件的模板 Tab 添加一个 `*_contact` 模板；系统会自动把该项纳入「优先送礼对象」下拉。

步骤：

1. 在 `src/data/characters.py` 中添加角色条目（示例）：

```py
characters["han_mei"] = {
    "zh": "寒梅",
    "en": "hanmei",
    "stars": 5,
}
```

2. 在程序的 Debug → 模板 Tab 中框选联络模板并保存（命名示例：`hanmei_contact`）。保存后模板会写入 `assets` 并可选生成枚举。

说明/验证：

- 模板保存后，若勾选“保存枚举”，工具会把 `hanmei_contact`（或相应名字）写入 `src/data/FeatureList.py`。如果未勾选，也可手动在 `FeatureList` 中添加 `hanmei_contact`。
- 系统通过 `get_contact_list_with_feature_list()` 将 `FeatureList` 中以 `_contact` 结尾的项与 `characters` 表中 `en+'_contact'` 键匹配，匹配到的中文名会自动出现在 Daily 任务页的「优先送礼对象」下拉里。

最小原则：仅新增角色信息与一个联络模板，保存后即可完成下拉配置，无需改任务代码或配置文件其他部分。
